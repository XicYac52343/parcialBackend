Intrucciones de Despliegue:

Requisitos:
	Descargar Librerías de Maven
	Versión Apache Netbeans IDE 19
	Versión Java 21
	Versión Xampp V3.3.0
	Versión TomCat Apache Tomcat/8.5.96



Paso a paso:

1. Ingrese al Ide apache
2. Dirigase a la opción "Team" en la parte superior
3. De clic en la opción "Git"
4. De clic en "Clone"
5. En la URL del respositorio pegue "https://github.com/XicYac52343/parcialBackend"
6. De next y eliga la rama main y next nuevamente
7. Recuerde crear la base de datos desde el admin de Mysql, la base de datos debe de llamarse "reservas" (Debe solo de crearla, sin ninguna tabla ni nada)
8. Y crear la conexión desde el apartado de services del IDE
9. Elegir Mysql y asignar el nombre de la base de datos "reserva"
10. O puede importar la base de datos adjunta a esta tarea (Pero también debe de crear la conexión desde el IDE y la importa desde el Admin Mysql)
11. Recuerde descargar las librerías de Maven, de lo contrario, el proyecto no funcionará por versión
12. Ya puede ejecutar el programa

URL PREDETERMINADA: http://localhost:8080/parcialBack/
PARA VISUALIZAR RESERVAS: http://localhost:8080/parcialBack/reservasJPAFront.jsp
